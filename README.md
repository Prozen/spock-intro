Demonstrates:

* the basic forms of Spock Tests
* mocking with Spock
* integration testing with spock

To see solutions, switch to the 'solutions' branch: 'git checkout solutions'

To run all tests, type 'gradle test'. *After* that, if you want to check your progress on branch coverage, run gradle testCoberturaReport.
The report ends up in build/reports/cobertura/
